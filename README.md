# CatchMeIFYOUCAN-TCP

## 프로젝트 개요
- 서버 - 클라이언트간 소켓 통신을 이해하고 활용
- 소켓 통신 기반 멀티 쓰레드 환경을 구축

## 서비스 소개
- 양방향 소켓 통신을 적극적으로 활용하는 아이디어 구상 → 게임 도출
- 시대의 문화생활 수요가 높아 티켓팅이 힘들다는 사실 파악
- 티켓팅 자리 선점 기반 1:1 대결 구조 게임
- GUI를 통한 직관적인 게임 환경 구성
- DB를 이용한 랭킹 시스템 구축

## 개발 환경
- 개발 도구 : intellij idea
- 언어 : Java
- 데이터베이스 : MySQL
